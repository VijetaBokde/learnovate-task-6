Perform all string operations and print line by line all the strings in single page



1) toLowerCase():Converts a string to lowercase letters.



2) toUpperCase():Converts a string to uppercase letters.



3) replace():Searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.



4) charAt():Returns the character at the specified index (position).



5) charCodeAt():Returns the Unicode of the character at the specified index.



6) concat():Joins two or more strings, and returns a new joined strings.



7)slice():Extracts a part of a string and returns a new string.



8)split():Splits a string into an array of substrings.



9)substr():Extracts the characters from a string, beginning at a specified start position, and through the specified number of character.



10)endsWith():Checks whether a string ends with specified string/characters.



11)fromCharCode():Converts Unicode values to characters.



12)includes():Checks whether a string contains the specified string/characters.



13)indexOf():Returns the position of the first found occurrence of a specified value in a string.



14)lastIndexOf():Returns the position of the last found occurrence of a specified value in a string.



15)localeCompare():Compares two strings in the current locale.
